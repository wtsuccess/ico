# ICO Presale Launch

Follow these steps:

## Step 1: Environment

First, move to the project directory.

Second, run command as follows:

`npm install`

## Step 2: Start

Run command as follows:

`npm run start`

## Step 3: Build

Run command as follows:

`npm run build`

npm and yarn are both good for this project.
